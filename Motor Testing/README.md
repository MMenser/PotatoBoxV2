# Starting
To start, open VSCode and install the PlatformIO extension. Open the folder 'Box Control System' in VSCode.

Build the project using the Check icon in the bottom right.
Library Error:
    Make sure you have the neccesary libraries installed. These are the include statements at the top of main.cpp. For example, this is where you would instll the Max31856 Library: https://registry.platformio.org/search?q=header:Adafruit_MAX31865.h
